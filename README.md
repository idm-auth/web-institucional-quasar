# Web Institucional - idm-auth.io

Site institucional do idm-auth.io desenvolvido com Quasar Framework (Vue 3 + TypeScript + Vite).

## Sobre o Projeto

Site institucional para apresentação do sistema idm-auth.io, incluindo informações sobre funcionalidades, documentação e recursos.

## Tecnologias

- Vue 3
- Quasar Framework v2
- TypeScript
- Vite
- Vue Router
- Pinia (State Management)
- Vue I18n (Internacionalização)
- Axios

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
quasar dev
```

## Qualidade de Código

### Lint

```bash
npm run lint
```

### Formatação

```bash
npm run format
```

## Build para Produção

```bash
quasar build
```

## Configuração

Veja [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js) para mais detalhes sobre customização.
